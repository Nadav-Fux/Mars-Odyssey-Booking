Hero Section & Mars Globe\n\n### Realistic Mars Globe (`RealisticMars.tsx`)\n\n**What it is:** A photorealistic rotating 3D Mars rendered entirely with CSS — no WebGL, no Three.js, no canvas.\n\n**How it works:**\n- A generated Mars texture (`mars-texture.webp`) is applied as a `background-image` on a circular `<div>`\n- `background-size: 300% 100%` creates a seamless tile, and `@keyframes mars-rotate` scrolls the `background-position-x` from `0` to `-300%` in an 80-second infinite loop\n- A clipped `border-radius: 50%` creates the sphere illusion\n- Multiple overlay `<div>` layers simulate 3D lighting:\n  - **Terminator shadow**: `linear-gradient(100deg, ...)` — simulates day/night boundary\n  - **Specular highlight**: `radial-gradient(ellipse 50% 45% at 32% 28%, ...)` — sun reflection upper-left\n  - **Limb darkening**: `radial-gradient(circle at 42% 48%, transparent 38%, ...)` — edges fade to black\n  - **Atmospheric rim**: `radial-gradient(circle, transparent 45%, rgba(255,80,20,0.04) 58%, ...)` — orange glow at edges\n  - **Fresnel edge highlight**: a `box-shadow: inset` on an outer ring\n- An outer `radial-gradient` haze creates the atmospheric glow effect\n\n**Technologies:** Pure CSS animations, CSS `radial-gradient` stacking, `border-radius: 50%`, `background-size` tiling\n\n**Easter Egg — Terraformed Mars:** Double-click the globe to activate. Applies `filter: hue-rotate(95deg) saturate(1.6) brightness(1.15)` turning red Mars into a blue-green habitable world. Unlocks the \"Terraformed\" achievement.\n\n### GSAP Scroll Zoom (`HeroSection.tsx`)\n\n**What it is:** As the user scrolls, Mars grows larger (scale 1 → 1.9), rotates slightly, and fades out — creating a cinematic \"approaching the planet\" effect.\n\n**How it works:**\n- GSAP `ScrollTrigger` with `scrub: 1.5` ties the animation to scroll position\n- A single GSAP timeline with two segments:\n  - 0%–55%: `scale: 1 → 1.6`, `rotation: 0 → 10`, `yPercent: 0 → -12`\n  - 55%–100%: `scale: 1.6 → 1.9`, `opacity: 1 → 0`\n- A separate timeline controls a `mars-glow-ring` element (radial gradient with blur) that pulses at 50% scroll\n- Hero text parallaxes out faster than Mars (`yPercent: 0 → -30`, `opacity: 1 → 0`)\n\n**Technologies:** GSAP 3 + ScrollTrigger, `scrub` mode, `@gsap/react` hook\n\n### Kinetic Title (`KineticTitle.tsx`)\n\n**What it is:** The main \"YOUR JOURNEY TO MARS\" headline with each word animating in with staggered vertical slide + opacity.\n\n**How it works:** GSAP `from()` with `stagger: 0.18` on `.hero-line` elements, each sliding from `y: 60` to `y: 0`.\n\n### Launch Countdown (`LaunchCountdown.tsx`)\n\n**What it is:** A live countdown timer in the hero overline pill, counting down to a future launch date.\n\n**How it works:** `setInterval` at 1s calculates `days:hours:minutes:seconds` from `Date.now()` to a target timestamp. Uses `tabular-nums` for flicker-free display.\n\n### Animated Counters (`AnimatedCounter.tsx`)\n\n**What it is:** Numbers that count up from 0 to their target value (54.6M km, 7 months, 2,847 booked).\n\n**How it works:** `requestAnimationFrame` loop with eased interpolation. Starts when `IntersectionObserver` detects the element is in viewport.\n\n---\n\n## 2. Solar System Flythrough\n\n### The Flythrough (`SolarFlythrough.tsx`)\n\n**What it is:** A scroll-pinned, 6-phase cinematic journey from deep space, past Earth, launching the ARES-7 ship, cruising through the void, and arriving at Mars.\n\n**How it works:**\n- Section is `height: 100vh` with `ScrollTrigger: { pin: true, end: '+=500%' }` — 5x the viewport height of scroll controls the entire sequence\n- A single master GSAP timeline with `scrub: 1` maps scroll position to animation progress\n- **6 phases** each control different elements:\n  1. **Deep Space (0–15%)**: Stars visible, \"SOLAR SYSTEM\" label fades out, sun glow appears\n  2. **Earth (15–35%)**: Earth scales from 0.04 → 1, moon orbit rotates\n  3. **Launch (32–48%)**: Ship appears, engine flare intensifies, ship accelerates right+up\n  4. **Transit (48–72%)**: Ship settles center, gentle bobbing (`sine.inOut`), speed lines, trajectory draws\n  5. **Mars Arrival (70–95%)**: Ship brakes (nose-up tilt), Mars scales in, ship enters orbit\n  6. **Fade (95–100%)**: Everything scales/fades, blackout overlay for section transition\n\n**Live HUD:** `tl.eventCallback('onUpdate')` reads `tl.progress()` to calculate distance, speed, and phase name in real-time.\n\n**Parallax Stars:** 3 star layers (120/60/30 stars) with different `xPercent`/`yPercent` movement speeds create depth.\n\n**Trajectory Path:** Triple-layer SVG `<path>` — wide glow blur, main gradient line, thin bright core. 5 animated particle dots travel along the path using SVG `<animateMotion>`.\n\n**Ambient Nebula:** Three `radial-gradient` layers prevent the section from appearing as pure black before scrolling begins.\n\n**Technologies:** GSAP ScrollTrigger (pin + scrub), SVG `<animateMotion>`, CSS gradients for nebula, staggered phase architecture\n\n### Planets (`SolarFlythrough.tsx > Planet()`)\n\n**What it is:** CSS-only 3D-looking planets (Earth, Mars).\n\n**How it works:** Stacked `<div>` layers:\n- Base: `radial-gradient(circle at 35% 30%, light, base, shadow)` — 3D sphere illusion\n- Atmosphere: outer `inset-[-6%]` div with radial gradient for atmospheric rim glow\n- Specular highlight: blurred elliptical white spot at upper-left\n- Limb darkening: dark edges radial gradient\n- Terminator: `linear-gradient(105deg, ...)` for day/night boundary\n- Children: continent patches, cloud bands, polar ice (additional gradient divs)\n\n---\n\n## 3.
