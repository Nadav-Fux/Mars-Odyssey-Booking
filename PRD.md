The critical rendering path is kept as small as possible:\n\n```\nindex.html\n├── <link preconnect> to fonts.googleapis.com + fonts.gstatic.com\n├── <link stylesheet> for Google Fonts (Orbitron, Space Grotesk, Exo 2)\n├── <script type=\"module\"> → main.tsx\n│   ├── App.tsx (React, Router, Providers, PageTransition)\n│   │   ├── LazyLoadErrorBoundary (wraps all routes — auto-retry on chunk failures)\n│   │   └── Index.tsx (lazy) → StarField, VerticalNav, HeroSection, NebulaBackground\n│   │       ├── DeferredMount(300ms) → Cursor, Scanlines, Toggles, BackToTop,\n│   │       │                          CrewChat, AchievementPanel, MissionLog,\n│   │       │                          MobileActionHub, PerformanceDashboard, AsteroidGameModal\n│   │       ├── DeferredMount(2s) → Terminal, CommandPalette, Audio, MarsClock, Overlays\n│   │       ├── DeferredMount(5s) → DustStorm, IncomingTransmission, ShipAI, DiscoveryHints\n│   │       └── LazySection → Destinations, Flythrough, Timeline, Stats, Gallery, ...\n│   └── Other pages (lazy) → ShipPage, CrewPage, MissionPage, ExplorePage, SimulatorPage\n└── GSAP loaded dynamically (not in main bundle)\n```\n\n### 10.2 Optimization Techniques\n\n| Strategy | Implementation |\n|----------|---------------|\n| **Font loading** | Moved from CSS `@import` (render-blocking waterfall) to HTML `<link>` with `preconnect` hints — eliminates 2-3 network hops |\n| **Image optimization** | Mars texture converted from PNG (462 KB) to WebP (63 KB) — 86% savings |\n| **Page-level code splitting** | All 6 pages are `React.lazy()` in `App.tsx` — including Index |\n| **Error recovery** | `LazyLoadErrorBoundary` wraps all routes — auto-retries failed dynamic imports (up to 2×), then hard-reloads. Prevents stale HMR cache crashes. |\n| **GSAP dynamic import** | `PageTransition` uses `import('@/lib/gsap')` instead of static import — GSAP stays out of the main bundle entirely |\n| **Section-level code splitting** | Every below-fold section uses `React.lazy()` + `Suspense` |\n| **Lazy mounting** | `LazySection` mounts children only when within 600px of viewport (IntersectionObserver) |\n| **3-tier deferred loading** | Tier 1: Eager (above-fold only). Tier 2: DeferredMount(300ms) for HUD controls. Tier 3: DeferredMount(2-5s) for overlays |\n| **Battery saver** | Global toggle disables GSAP animations, particles, CRT effects |\n| **Multi-page split** | Heavy content distributed across 6 routes instead of 1 mega-page |\n| **rAF animations** | 3D tilt, cursor effects, and count-up numbers use requestAnimationFrame, not React state |\n| **will-change hints** | Applied to transform-heavy elements |\n| **ScrollTrigger refresh** | Triple-layer refresh (PageTransition, useGSAP, delayed useEffect) ensures GSAP pin calculations are always correct after page transitions |\n| **GSAP scoping** | All `useGSAP` calls use `{ scope: ref }` to prevent selector leaks between pages |\n\n---\n\n## 11.
